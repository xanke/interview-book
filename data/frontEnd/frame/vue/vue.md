### 组件之间的传值

- 父组件与子组件传值：通过标签定义，子组件通过 props
- 子组件向父组件传递数据：子组件通过 $emit，父组件通过 @event

### v-show 和 v-if 指令的区别

- v-show 指令是通过修改元素的 display 的 CSS 属性让其显示或者隐藏
- v-if 指令是直接销毁和重建 DOM 达到让元素显示和隐藏的效果
